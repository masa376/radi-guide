# RadiGuide（ラジガイド）

## 概要

放射線技師として医療現場で勤務した経験をもとに設計・開発した、患者向け放射線検査説明Webアプリケーション。
検査前の準備・検査の流れ・よくある質問をわかりやすくまとめることができ、患者が安心して検査を受けられるよう支援します。


## 使用技術

### バックエンド
- PHP：8.3
- Laravel：11
- MySQL：8.4

### フロントエンド
- React
- TypeScript
- Alpine.js
- TailwindCSS

### インフラ・開発環境
- Docker / Laravel Sail：ローカル開発環境
- Git / GitHub：バージョン管理


## 機能一覧

### 一般ユーザー向け（ログイン不要）

**トップページ**
- カテゴリ別検査一覧表示
- キーワード検索（トップページから直接検索可能）

**検査一覧・検索**
- React + TypeScriptによるリアルタイム検索
  - キーワード入力で即座に結果更新
  - カテゴリ絞り込み
  - Laravel APIとの連携
- 検索結果件数の表示

**検査詳細ページ**
- 検査の概要・目的・流れ・注意事項
- 検査所要時間のタイムライン表示
- **Reactによるインタラクティブなチェックリスト**
  - チェック状態をlocalStorageに保存
  - アニメーション付き進捗バー
  - リセットボタン
- 印刷用ページの作成

**よくある質問**
- Alpine.jsによるアコーディオン表示
- 全般FAQ・検査別FAQの分類表示

### 管理者向け（ログイン必要）
- カテゴリ管理（CRUD）
- 検査記事管理（CRUD・公開/非公開管理）
- チェックリスト管理（CRUD・必須/任意設定）
- FAQ管理（CRUD・全般/検査別設定）
- 一般ページから管理者ログインボタンでアクセス可能


## 設計ポイント

### 検査前チェックリストの必須/任意区分
放射線検査では検査前の準備が重要です。
CT造影剤使用時の絶食・MRIの金属除去など、必須項目と任意項目を色分けして表示し、患者が見落とさないように設計しました。

### 注意事項の強調表示
造影剤アレルギーなど重要な注意事項は黄色背景で強調表示しています。

### FAQ の全般/検査別分類
「予約方法」などの全般的な質問と「CT検査に痛みはありますか？」などの検査固有の質問を分けて管理・表示できます。

## データベース設計

```
categories   - 検査カテゴリ（X線・CT・MRI・エコーなど）
examinations - 検査記事（メインコンテンツ・公開管理）
checklists   - 検査前チェックリスト項目
faqs         - よくある質問（全般/検査別）
```


### 工夫した点
- 管理者側（/admin）と一般ユーザー側をControllerのディレクトリで明確に分離しました
- React + TypeScript + Laravel APIの構成でリアルタイム検索を実現しました
- localStorageでチェックリストの状態を保存し、ページを再訪問しても状態が保持されます
- Alpine.jsとReactを用途に応じて使い分けました
- 印刷用CSSでブラウザ印刷に最適化したページを作成しました

## ローカル環境での起動方法

```bash
# リポジトリのクローン
git clone https://github.com/masa376/radi-guide.git
cd radi-guide

# 環境変数の設定
cp .env.example .env

# Sailの起動
sail up -d

# 依存パッケージのインストール
sail composer install

# アプリケーションキーの生成
sail artisan key:generate

# マイグレーションの実行
sail artisan migrate

# フロントエンドのビルド
sail npm install
sail npm run build

# シーダー実行（デモデータ投入）
sail artisan db:seed

# フロントエンドのビルド
./vendor/bin/sail npm install
./vendor/bin/sail npm run build

```
